# Upload GitHub Pages - Inline Assets Version

Versi ini dibuat supaya gambar tetap muncul walaupun folder `assets/` tidak ter-upload.

Yang perlu di-upload minimal:
- semua file `.html`
- semua file `.pdf` jika ingin update PDF juga

Folder `optional-png-backup/` tidak wajib di-upload karena gambar sudah ditanam di dalam HTML.

Cara paling aman:
1. Buka folder ini.
2. Select semua file di root folder ini, kecuali folder `optional-png-backup` juga boleh tidak dipilih.
3. Upload ke root repository GitHub Pages.
4. Commit changes.
5. Tunggu 1-5 menit.
6. Buka link dengan `?v=inline1`.
